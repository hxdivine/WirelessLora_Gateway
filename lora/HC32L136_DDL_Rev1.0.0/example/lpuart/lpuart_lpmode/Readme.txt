功能描述
================================================================================
本样例主要涉及低功耗串口中断收发数据样例

说明：
配置PA00、PA01为LPUART收发端口
配置波特率9600bps
使能LPUART中断
使能LPUART收发功能
================================================================================
测试环境
================================================================================
测试用板:
HC32L136_STK
---------------------
（需要填）

辅助工具:
---------------------
串口线

辅助软件:
---------------------
串口调试工具

================================================================================
使用步骤
1、重新编译代码运行
2、串口调试工具配置波特率9600bps、偶校验、1stop
3、PD04按下按键，系统进入深度休眠
4、PC串口发送10字节数据，唤醒系统
5、发送完成后，收到10字节下发数据+0x11
