功能描述
================================================================================
本样例主要展示低功耗Timer的计数功能。
说明：
本样例对展示低功耗Timer在低功耗模式下对外部时钟进行计数，直到计数到溢出值时产生中断，
退出低功耗模式并将LED控制信号翻转，中断处理结束后回到低功耗模式，继续计数直到下次中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）准备外部时钟源信号，例如可以是示波器的输出校准信号
4）运行；
5）程序会等待user key按下，只有user key按下之后会进入超低功耗模式。
6）在超低功耗模式下，若PB04输入外部时钟信号，LPTimer会从0xE000进行计数，直到溢出产生中断

================================================================================
注意
================================================================================
1.为保证正常计数，外部时钟源须小于Timer运行时钟的一半。

    注：若芯片处于低功耗模式，则芯片无法使用SWD进行调式和下载功能。故如需要再次下载
调试其他程序，需要将芯片复位，并且复位后下载程序前不能按下user key。
