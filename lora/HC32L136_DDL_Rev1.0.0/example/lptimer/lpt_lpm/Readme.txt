功能描述
================================================================================
本样例主要展示低功耗Timer的计数功能。
说明：
本样例对展示低功耗Timer在低功耗模式下的定时器功能，使用RCL时钟从0到0xFFFF，溢出产生中断，
退出低功耗模式并将LED控制信号翻转。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行；
4）程序会等待user key按下，只有user key按下之后LPT运行并且进入超低功耗模式。
5）在超低功耗模式下，LPT继续运行，计数溢出后产生中断，退出超低功耗模式、翻转LED端口的输出电平

================================================================================
注意
================================================================================
1.若芯片处于低功耗模式，则芯片无法使用SWD进行调式和下载功能。故如需要再次下载
调试其他程序，需要将芯片复位，并且复位后下载程序前不能按下user key。
