功能描述
================================================================================
本样例主要展示PCA的捕获功能。
说明：
本样例对展示PCA对VC1的输出电平上升/下降沿的捕获功能，捕获到沿时产生中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器连接PA07(VC1_OUT)查看VC1的输出波形；
4）用跳线连接PB14
4）运行；
5）向PB14输入上升沿/下降沿信号，可以在调试窗口查看变量‘u16CapData’
   和变量‘u16CntData’的值。示波器可以查看VC1的比较结果输出。



================================================================================
注意
================================================================================
1）当被捕获电平的边沿变化时间大于PCA的CNT时间，可以设置计数溢出中断使能，
   在溢出中断内统计溢出的次数。然后当边沿捕获发生时，再与PCA的捕获值求和。
