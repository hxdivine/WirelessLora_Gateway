功能描述
================================================================================
本样例主要涉及时钟模块周期中断实例

说明：
计数初值写入计数寄存器
配置4s周期中断
启动计数
仿真是否正常进入中断，端口观察PD05是否4s高低电平变化

用万用表的电流档接入Demo板J4接口，按下user键，进入超低功耗模式，LED灯灭的时候可以观察MCU在RTC运行时候和XTL使能时候的功耗。
