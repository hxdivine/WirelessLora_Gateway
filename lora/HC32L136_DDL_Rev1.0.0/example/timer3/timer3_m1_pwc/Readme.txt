功能描述
================================================================================
本样例主要展示Timer3的脉宽测量功能。
说明：
本样例使用timer3的PWC循环脉宽测量功能。
================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）向PA08（TIM0 CHA）输入一个PWM波，根据程序设定，测量PWM的上升沿到下降沿的计数值。
