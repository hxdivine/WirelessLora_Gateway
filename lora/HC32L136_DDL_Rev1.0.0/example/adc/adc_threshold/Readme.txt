功能描述
================================================================================
本样例主要展示ADC 扫描连续采样区间值比较示例，使用中断方式获取ADC运行状态和ADC的采样值。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）在三个ADC输入PIN(PA00 PA02 PA05)输入不同的电平。
3）调整ADC输入PIN(PA00)电平值进入不同的区间。
4）在ADC中断内读取转换值，并设置转换完成Flag。
5）在ADC区间触发中断内读取转换值，并设置转换完成Flag。
6）在主循环里面等待转换完成，延时，然后再次开启扫描转换
