本样例主要展示ADT VC中断刹车功能。
说明：
    

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）PB14作为VC1的P输入，PA00作为VC1的N输入,PA02作为VC1的输出（仅用于示波器观察）
2）PB14接GND，PA00接VCC，
3）打开工程编译并运行
4）示波器观测PA08(IA0)和PA07(IB0)端口输出波形，开始运行时端口输出正常PWM波形，
   PA02输出低
5）PB14接VCC，PA00接GND，此时PA02输出高，PWM停止（按照刹车时预设电平输出）


================================================================================
注意
================================================================================
VC触发ADT刹车， 注意VC中断的时间，VC中断结束后刹车即失效，PWM恢复输出。
