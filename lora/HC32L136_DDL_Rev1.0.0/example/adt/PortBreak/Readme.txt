功能描述
================================================================================
本样例主要展示ADT端口刹车与软件刹车功能。
说明：
    

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）连接板子PB01与PB11，示波器连接PA08,PA07和PB01
2) 打开工程编译并运行
3）PA08与PA07输出PWM
4）端口极性与端口刹车配置相同时，PWM被关断，寄存器窗口可以看到端口刹车标志被置起
5）软件刹车位置起时，PWM被关断，寄存器窗口可以看到软件刹车标志位被置起


================================================================================
注意
================================================================================
1）刹车状态：PA08 Low level，PA07 High level
2）PB01输出刹车信号到PB11：
   第一次刹车：High level 有效刹车
   第二次刹车：Low level有效刹车
   第三次刹车：软件刹车
