功能描述
================================================================================
本样例主要展示PCNT 正交编码计数功能。
说明：
本样例对展示PCNT 正交编码计数产生中断，并且工作在低功耗模式下。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L136_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）外部输入信号到PB05和PB07；
4）运行；
5）观察LED信号线与输入脉冲关系，输入一个脉冲信号，计数一次同时LED灯翻转闪烁。
